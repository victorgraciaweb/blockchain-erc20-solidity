# blockchain-erc20-solidity
ERC20 Smart Contract developed with solidity, Ganache and Truffle in Ethereum blockchain with Test.

How to run?
--

1. Install Truffle

   `npm install truffle`

2. Compile and migrate Smart Contract

   `truffle migrate`

3. Open Truffle console

   `truffle console`

4. Test

   `truffle test`
